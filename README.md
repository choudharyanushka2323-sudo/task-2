# Student Management System (Java)

## 📌 Internship Task 2

A simple console-based **Student Management System** built in Java to demonstrate the use of **loops** and basic control structures for managing student records.

## 📖 Description

This program allows users to manage a small set of student records through a menu-driven console interface. It uses loops (`for`, `while`, `do-while`) to:

- Add student records
- Display all student records
- Search for a student
- Calculate average marks / grades
- Repeat the menu until the user chooses to exit

## 🚀 Features

- Add new student (name, roll number, marks)
- View list of all students
- Search student by roll number / name
- Calculate and display average marks
- Loop-driven menu system for continuous interaction until exit

## 🛠️ Tech Stack

- **Language:** Java
- **Concepts used:** Loops (for/while/do-while), Arrays/ArrayList, Conditional statements, Methods
